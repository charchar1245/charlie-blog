Regression

Housing Prices Predictor

A PyTorch-based regression model that predicts house sale prices in USD using 81 engineered features from the Kaggle "House Prices – Advanced Regression Techniques" dataset. Achieves an R² of 0.97 on test data.

Classification

Diabetes Classification Model

A neural network classifier trained on 100,000 patients with 9 clinical features to assist in diabetes diagnosis. Uses dropout, pos_weight, and sigmoid outputs, reaching 60% precision, 89% recall, and a 70% F1 score.
